# 表格转excel
可以将页面中的表格转成excel并导出

## 可以导出的类型
1. 只能是table结构
2. 默认导出页面上第一个表格，可导出指定表格
3. 导出的表格，可以是一个原生表格（th和tr在一个table内）
4. 可以是大部分ui库那样分成两个，th和tr分别在不同的table内

## 使用方法
1. 导入插件后，刷新页面会加载插件面板，点击导出数据就可以使用了（推荐）
2. 右键导出，只能导出默认第一个表格（可在background.js中开启）
3. 右键可控制插件显示隐藏
